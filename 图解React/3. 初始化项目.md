## 初始化项目
#### 创建项目
选定一个空文件夹，执行创建react项目命令
```
create-react-app project-name
```
###### 创建TS项目
```
npx create-react-app my-app-name --template typescript
```

![](./初始化项目/图二.jpg)

#### 运行项目
打开创建的项目```或者```进入到创建的项目执行命令
```
yarn start
```
